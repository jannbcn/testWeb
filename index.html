Web de Ejemplo

comandos GIT: git init git status -> para ver el estado de los ficheros git log -> para poder ver los commits realizados git log --oneline -> si añadimos --oneline podremos ver los commits en una sola linea y el codigo de los commits en caso de necesitarlos para un reset --hard. git commit -m "" -> el -m es para añadir un comentario git checkout "código de commit" -> para volver a otro commit anterior sin perder los cambios hechos en caso de necesitarlos, es más recomendable que el reset
