comandos GIT: git init git config --global user.name "Your Name" -> para poder cambiar tu nombre de autor git config --global user.email you@example.com -> para poder cambiar tu email de autor git status -> para ver el estado de los ficheros git log -> para poder ver los commits realizados git log --oneline -> si añadimos --oneline podremos ver los commits en una sola linea y el codigo de los commits en caso de necesitarlos para un reset --hard. git commit -m "" -> el -m es para añadir un comentario git checkout + código de commit -> para volver a otro commit anterior sin perder los cambios hechos en caso de necesitarlos, es más recomendable que el reset git checkout + nombre rama -> para cambiar a la nueva rama creada git branch + nombre rama -> para crear una nueva rama git merge + nombre rama -> para poder juntar el código que has hecho en otra rama y actualizarlo, este comando se realiza desde donde desees realizar esta unión branches -> el principal es la branch MASTER, a partir de aquí para no afectar a la rama principal, creamos una rama nueva, que es un universo paralelo para poder realizar los cambios que desee sin afectar al codigo principal y luego cuando el código sea estable, ya podría actualizar el código principal. HEAD -> en donde te encuentras editando
